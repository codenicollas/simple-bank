# Sistema Bancário - POO (Unisinos)

Trabalho prático de Java focado em herança e polimorfismo para a disciplina de Programação Orientada a Objetos.

### 💻 Funcionalidades
* Gerenciamento de contas Corrente, Poupança e Investimento.
* Operações de depósito, saque (com lógica de cédulas) e juros.
* Geração de extrato detalhado com histórico de saldos.

Nota: Este projeto foi desenvolvido para fins acadêmicos sob a orientação do Prof. Márcio Miguel Gomes.
